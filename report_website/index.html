Datasets

Why These Datasets?

All three datasets are from UCI ML Repository — collected from real hardware under real conditions.

📡

Real Sensors

Data from actual robots and smartphones — not simulated. Real noise, real variability, real edge cases.

🧩

Diverse Structures

Pre-engineered vectors · raw time-series · spatial sensor arrays — each demands different preprocessing.

⚖️

Class Imbalance

Navigation has 40% Forward vs 6% Slight-Left. Failure detection has rare classes. Tests real-world robustness.

📈

Graded Difficulty

HAR (~95%) → Navigation (~99% with NL boundaries) → Failure (small sample, 11 classes).

Action Typical sensor pattern

Reinforcement Learning

🚕 Taxi-v3 Autonomous Agent

No labelled data. The agent learns entirely from reward signals by interacting with the environment.

Environment — Taxi-v3

Gymnasium · 5×5 grid world

Property Value
State space 500 discrete states
Action space 6 (N·S·E·W·Pick·Drop)
Reward +20 drop · −10 illegal · −1 step
Challenge Sparse reward + delayed feedback
ε-greedy 1.0 → 0.05 (decay 0.995 / ep)
γ discount 0.99

🗃️

Replay Buffer

3,000 transitions. Random mini-batch (size 32) breaks temporal correlations in training.

🎯

Target Network

Separate frozen copy, synced every 25 episodes. Prevents oscillating Q-value targets.

⚡

Train Every 4 Steps

Skips gradient update 3/4 steps — reduces CPU overhead by ~4× vs. training every step.

💡

Why Tabular Wins Here

500 states fits perfectly in a Q-table. DQN shines in continuous/huge state spaces like Atari or robot joints.
